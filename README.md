# Hand-Tracked 3D Particles

Interactive 3D particle system using **Three.js** and **MediaPipe Hands**.

## Features
- Real-time hand tracking via webcam
- Gesture-based particle **size, color, and shape control**
- Multiple particle shapes: sphere, heart, star, Saturn, firework
- Fully browser-based (no installation needed)
- Works on Chrome/Edge/Firefox
